Магическим квадратом порядка n называется квадратная таблица размера n×n, составленная из всех чисел 1,2,3,…,n^2  так, что суммы по каждому столбцу, каждой строке и каждой из двух диагоналей равны между собой. 

С клавиатуры вводится натуральное число n — количество строк и столбцов в матрице, затем элементы матрицы: n строк, по n чисел в каждой, разделённые пробелами.

Напишите программу, которая проверяет, является ли заданная квадратная матрица магическим квадратом.
